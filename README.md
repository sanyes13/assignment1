## Контрольные вопросы

1. [cite_start]**Динамический vs Статический массив**: Статический имеет фиксированный размер при компиляции, динамический выделяется в "куче" во время работы[cite: 14].
2. **Указатель**: Переменная, хранящая адрес памяти; [cite_start]необходима для управления динамическими объектами[cite: 15].
3. [cite_start]**Освобождение памяти**: Предотвращает утечки памяти (memory leaks), которые могут привести к падению системы[cite: 16].
4. [cite_start]**Параллельная обработка**: Позволяет распределять задачи между ядрами процессора для ускорения расчетов[cite: 17].
5. [cite_start]**#pragma omp parallel for**: Директива, которая автоматически делит итерации цикла между потоками[cite: 18].
6. [cite_start]**Reduction**: Механизм для объединения локальных результатов каждого потока в одну итоговую переменную без ошибок доступа[cite: 19, 20].
7. [cite_start]**Замедление параллельной версии**: Может возникать из-за накладных расходов на создание потоков или если объем данных слишком мал[cite: 21].
